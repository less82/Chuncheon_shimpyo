# 01. 제품·화면 기준

> 제품 범위와 시민 화면의 단일 기준이다.

## 제품 정의

쉼표 정류장은 출발 위치와 목적지를 받아 **목적지행 버스의 도착예정시간**을 알려주고, 정류장 상태를 시민–행정 업무로 연결하는 서비스다.

- 시민 가치: 탈 정류장·방면·버스 도착시간을 빠르게 확인한다.
- 반복 가치: 출발–목적지 쌍을 즐겨찾기로 다시 조회한다.
- 행정 가치: 시민 신호를 정류장 단위로 묶어 확인·갱신한다.
- 1차 사용자: 복잡한 조작과 스크롤이 어려운 고령 이용자.

조회 기록은 실제 승차량이나 고령자 수요가 아니다.

## 진입 화면

| 진입 | 대상 | 첫 화면 |
|---|---|---|
| 설치 앱 | 반복 이용자 | `버스`, `정류장`, 즐겨찾기 |
| 공통 QR | 첫 방문자 | 할 일 선택 |
| 관리자 웹 | 담당자 | 인증 후 업무 목록 |

설치 앱에 QR 스캔, 로그인 안내, 서비스 소개를 반복하지 않는다.

## 시민 앱 흐름

| 단계 | 화면 | 완료 조건 |
|---:|---|---|
| 1 | 메인 | 즐겨찾기 또는 `버스` 선택 |
| 2 | 장소 입력 | 출발 위치와 목적지 입력 완료 |
| 3 | 장소 확인 | 지도·주소를 보고 사용자가 좌표 확정 |
| 4 | 버스 조회 | 목적지행 버스와 올바른 승차 지점 산출 |
| 5 | 도착정보 | 가장 빠른 버스 도착시간 확인 |
| 6 | 후속 행동 | 즐겨찾기 또는 정류장 상태 알리기 |

### 장소 입력

- 출발지와 목적지는 정류장이 아니라 장소다.
- 음성 버튼은 `출발지 말하기`, `목적지 말하기`로 쓴다.
- 다시 누르면 듣기를 중단한다.
- 듣는 동안 버튼 색·상태 문구를 유지한다.
- 오류는 해당 입력 바로 아래에만 표시한다.
- `지금은 말할 수 없어요`를 누르면 직접 입력으로 전환한다.
- 입력 글자 수만으로 다음 단계로 넘기지 않는다.
- 검색 후보가 없거나 춘천 밖이면 다시 입력하도록 한다.

### 결과 화면

정보 우선순위는 다음과 같다.

1. 버스 도착예정시간
2. 버스 번호
3. 탈 정류장명·방면
4. 출발 위치에서 정류장까지 도보시간
5. 실시간 정보 생성시각
6. 즐겨찾기와 정류장 상태 알리기

첫 화면에는 가장 빠른 후보와 다음 후보만 둔다. 노선도·전체 경유지·장황한 길 설명은 넣지 않는다. 총 소요시간과 목적지 도착예상은 검증된 경로 API가 값을 줄 때만 표시한다.

실시간 조회가 실패하면 `실시간 도착정보를 불러오지 못했어요`라고 쓴다. 배차간격을 도착예정시간처럼 보여주지 않는다.

### 즐겨찾기

장소 하나가 아니라 다음 이동 정보를 함께 저장한다.

- 출발 위치
- 목적지
- 탑승·하차 정류장
- 버스와 방면

즐겨찾기를 열 때 경로와 도착시간을 다시 조회한다. 저장 당시 도착시간은 재사용하지 않는다.

### 정류장 상태 알리기

- 길찾기 결과에서 진입하면 탑승 정류장을 그대로 넘긴다.
- 별도 진입이면 위치·정류장명·4자리 번호로 대상을 확인한다.
- 시민 입력은 사실 확정이 아니라 검수 전 신호다.
- AI 구조화 결과는 제출 전에 사용자가 확인·수정한다.
- 공식 연계 전에는 `민원 접수`라고 표시하지 않는다.

## 행정 흐름

```text
시민 신호 → AI 구조화·중복 묶기 → 공식자료 대조
→ 담당자 확인 → 현장 결과 확정 → 시민 화면 갱신
```

AI는 시설·문제·지속성 후보를 만든다. 정류장 ID, 시설 상태, 공사와 예산은 사람이 확정한다.

## 화면 규칙

- 실제 사용영역은 안전영역을 뺀 높이다.
- 기준 화면은 320×461px이며 핵심 흐름에 세로 스크롤을 두지 않는다.
- 글자·간격·버튼 높이는 `clamp()`로 점진적으로 변하게 한다.
- 글꼴은 나눔고딕, 기본 글자는 `#333333`이다.
- 키 컬러는 춘천 도시브랜드 PANTONE 299C 근사값 `#00A3E0`이다.
- 안전영역 미리보기만 노란색으로 구분한다.
- 주요 터치 영역은 48px 이상으로 한다.
- 꺾쇠와 조작 아이콘은 `#333333`을 사용한다.
- 아이콘은 조작 의미가 분명할 때만 쓰고 한 화면에서 한 제품군을 사용한다.
- 헤더에는 불필요한 실선·그림자·`메인` 문구를 넣지 않는다.
- 상태는 색만으로 표현하지 않는다.
- 오류·경고·성공색은 브랜드색보다 채도를 낮춘다.
- OS 안전영역과 앱 콘텐츠가 겹치지 않는지 iOS·Android 실기기에서 확인한다.

## 금지

- 동명 정류장을 이름만으로 자동 선택
- 맞은편 정류장 시설 합산
- 검색 결과 임의 보정
- 근거 없는 시간·경로 생성
- 제보량을 설치순위나 수혜인원으로 표현
- 로그인·설명·장식 아이콘으로 첫 화면 점유
