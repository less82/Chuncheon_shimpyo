# 쉼표 정류장 — 고령 사용자 테스트 (양식 · A10)

> 본선 전 필수(설계 §5). 60대 이상 3~5명 대상. 이 문서는 **기록 양식**이며, 실제 테스트는 사람이 수행한다.
> 목적: "앉아서 기다리는 길"과 QR 등록이 실제 고령자에게 통하는지 검증하고, 발견 문제를 UI에 반영해 발표 1장으로.

## 준비물
- 실기기(안드로이드 1대 + 아이폰 1대 권장), 배포본 URL(A13 후) 또는 `npm run dev` + 폰 접속.
- 정류장 QR 포스터 출력물 1장(A8/PrintPoster).
- 관찰자 1명(기록), 진행자 1명.

## 태스크 5종 (각 태스크: 성공여부 / 소요시간 / 오조작 횟수 / 관찰 메모)
1. **QR 등록** — 포스터 QR을 폰 카메라로 찍어 그 정류장을 즐겨찾기에 넣기.
2. **목적지 선택** — 즐겨찾기에서 목적지(예: 병원)를 탭으로 골라 경로 보기.
3. **경로 카드 이해** — "어느 정류장으로 가시겠어요?"에 카드를 보고 답하기(편안 경로 이해도).
4. **글자·버튼 평가** — 글자 크기·버튼 크기가 편한지(편함/보통/불편).
5. **오프라인 반응** — 비행기모드에서 앱을 열었을 때 반응(당황/이해/무반응).
