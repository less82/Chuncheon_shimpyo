## 2. 핵심 태스크

1. 포스터의 정류장 진입 QR을 촬영한다.
2. 화면의 정류장이 현재 정류장인지 확인한다.
3. 목적지를 음성으로 입력한다.
4. 앱이 알아들은 목적지가 맞는지 확인한다.
5. 탈 버스와 도착예정시간을 말해 달라고 요청한다.
6. 다른 방향 정류장과 혼동하지 않았는지 확인한다.
