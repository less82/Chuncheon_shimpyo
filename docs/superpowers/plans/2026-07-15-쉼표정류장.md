# 쉼표 정류장 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 춘천시 버스정류장의 그늘·의자·조명·도착안내기 유무를 3상태(있음/없음/미확인)로 보여주는 고령자용 시민 웹앱과, 쉘터 설치 후보를 조건 필터로 뽑는 행정 대시보드를, 실데이터 파이프라인 위에서 오프라인 작동하도록 만든다.

**Architecture:** Python/pandas 파이프라인이 6개 원본 CSV를 결합해 단일 `public/data/stops.json`을 생성 → Vite+React+TS 앱이 이를 정적으로 읽어 시민앱(`/`)과 대시보드(`/admin`) 두 라우트로 렌더. 서버 0개, Leaflet+OSM 지도, PWA로 오프라인 캐시.

**Tech Stack:** Python 3, pandas / Vite, React 18, TypeScript, React Router, Leaflet, Zustand(경량 상태), vite-plugin-pwa / vitest, pytest.

## Global Constraints

- 시설 상태는 `"yes" | "no" | "unknown"` 유니온만. **근거 없이 `"no"`를 생성하는 코드 경로 금지.** 기본값은 항상 `{status:"unknown", source:"none"}`.
- 배지 문구: 로드뷰 출처는 `"로드뷰 확인 (촬영 YYYY.MM)"`. 그 외는 `"○○대장 기준"`. **문자열 "현장 확인" 은 코드·화면 어디에도 넣지 않는다.**
- 대시보드는 **합성 점수(가중치 점수) 표시 금지.** 조건 필터와 실측 수치만.
- 수요(승차량)에는 항상 **"양방향 합산 기준"** 표기를 동반.
- 마스터 조인 키 = 위치정보 `관리번호`(250xxx). 정류장 이름으로 마스터를 조인하지 않는다(수요 브리지 제외).
- 네트워크 없이 데모가 돌아가야 한다. 지오코딩·TAGO 등 네트워크 의존은 빌드 시점에만, 결과는 `stops.json`/캐시에 구워 런타임은 정적.
- 좌표계: WGS84 위경도. 거리 계산은 Haversine(m).
- 인코딩: 모든 원본 CSV는 **cp949**로 읽는다.
- 데이터 원본 폴더: `data/` (프로젝트 루트). 파이프라인 출력: `app/public/data/`.
- 커밋은 각 태스크 끝에서. 커밋 메시지 끝에 `Co-Authored-By: Claude Opus 4.8 (1M context) <noreply@anthropic.com>`.

**정류장 마스터 스키마 (`stops.json` 배열 원소) — 전 태스크 공통 계약:**

```ts
type Facility = "yes" | "no" | "unknown";
type Source   = "roadview" | "bench_registry" | "shade_registry" | "light_registry" | "none";
interface FacilityInfo { status: Facility; source: Source; capturedAt?: string; }
interface Demand { byHour: number[]; total: number; aggregatedBidirectional: boolean; matchedName: string; }
interface Stop {
  id: string; stopNo: string; name: string; lat: number; lng: number;
  routes: string[];
  facilities: { shade: FacilityInfo; seat: FacilityInfo; light: FacilityInfo; sign: FacilityInfo; };
  demand?: Demand;
  headwayMin?: number;
}
interface StopsFile { generatedAt: string; cityCenter: {lat:number; lng:number}; stops: Stop[]; }
```

`cityCenter` = 춘천시청 `{lat:37.8813, lng:127.7300}` (위치권한 거부 시 폴백).
