# 쉼표 정류장 v2 확장 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`) syntax.

**Goal:** 기존 앱에 ①실시간 버스 도착(TAGO) ②시설 필터+지도강조+도보경로 ③QR 공유강화 ④목적지 길찾기(환승 포함)를 얹되, 네트워크·키 없이도 폴백으로 동작하게 한다.

**Architecture:** 파이프라인이 `routes.json`(노선별 정류장 순서 그래프)과 (키 있으면)정류장↔TAGO nodeid 매핑을 추가 생성. 프론트는 TAGO 실시간/OSRM 도보경로를 시도하고 실패 시 배차간격/직선거리로 폴백. 목적지 길찾기는 route 그래프 BFS(환승≤2).

**Tech Stack:** Python/pandas, React19+TS, Leaflet, zustand, OSRM foot API, TAGO API, 경량 QR 라이브러리(qrcode). 기존 vitest/pytest.

## Global Constraints

- v1 정직성 규칙 전부 유지: 시설 3상태(근거 없는 "no" 금지), "현장 확인" 금지, 합성 점수 금지, 수요 "양방향 합산" 표기.
- **키보드 0의 의미**: 키보드 기능을 없애는 게 아니라, 키보드 없이도 모든 핵심 흐름(공유·목적지선택·시설선택)이 탭/스캔만으로 완결돼야 한다. 타이핑은 선택적 지름길.
- 접근성 하한: 폰트 ≥18px, 터치타깃 ≥48px, 아이콘+한글 병기, 대비 WCAG AA. 기존 `app/src/styles/tokens.css` 재사용.
- **오프라인 하드 제약**: 실시간(TAGO)·도보경로(OSRM)·경로탐색은 네트워크/키 의존 → 항상 폴백 존재, 무한 스피너 금지. 폴백 문구는 "약 N분", "직선거리 기준" 등 정직 표기. 거짓 "실시간" 금지.
- TAGO 키는 `app/.env`의 `VITE_TAGO_KEY`(런타임), 파이프라인 매핑용은 동일 키를 환경변수로. 키 없으면 관련 빌드/호출 skip → 폴백. 키를 코드/커밋에 넣지 않는다.
- 마스터 키 = 관리번호(stops.json `id`). routes.json도 관리번호로 정류장을 참조.
- 거리=Haversine(m), 도보속도=80m/분(기존 대안정류장과 일관).
- 커밋: 태스크마다, `git add <명시 경로>`(전체 add 금지), 메시지 끝 `Co-Authored-By: Claude Opus 4.8 (1M context) <noreply@anthropic.com>`.

**신규 타입 (전 태스크 공통):**
```ts
// app/src/types/route.ts
export interface RouteInfo { routeId: string; routeNo: string; stops: string[]; } // stops = 관리번호 순서
export interface RoutesFile { generatedAt: string; routes: RouteInfo[]; }
// app/src/types/trip.ts
export interface TripLeg { routeNos: string[]; boardStopId: string; alightStopId: string; }
export interface TripOption {
  boardStopId: string; walkMin: number; walkReal: boolean;
  legs: TripLeg[]; transferStopId?: string; directBus: boolean;
}
```
