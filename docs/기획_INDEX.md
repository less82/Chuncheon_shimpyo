내용은 해당 카테고리 문서 한 곳에서만 수정한다. 파생 문서에 같은 설명이나 작업 목록을 복사하지 않는다.

## 사실 기록

- `데이터_현황.md`, `데이터_검증.md`: 2026-07-15 산출물 기록
- `보안점검.md`, `QA_체크리스트.md`: 당시 코드 점검 기록
- `specs/`, `superpowers/plans/`, `OPUS_핸드오프.md`: 과거 설계·계획·검토 기록

사실 기록은 당시 상태를 재현할 때만 사용한다. 현재 요구사항은 아니다.

## 호환 문서

기존 문서 링크는 삭제하지 않고 통합 문서로 보내는 안내만 남겼다. 새 내용은 `현재/`에만 작성한다.
